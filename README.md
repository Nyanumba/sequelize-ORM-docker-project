# Customer Order Management API

A backend RESTful API for managing customers, orders, and products.  
Built with Node.js, Express, Sequelize ORM, and PostgreSQL, and fully containerized using Docker.

---

## Features

- CRUD operations for:
  - Customers
  - Orders
  - Products
- Database relationships:
  - One-to-Many (Customer → Orders)
  - Many-to-Many (Orders ↔ Products)
- Sequelize ORM with:
  - Migrations (database version control)
  - Seeders (sample data)
- Dockerized environment for easy setup

---

## Tech Stack

- Node.js
- Express.js
- Sequelize ORM
- PostgreSQL
- Docker & Docker Compose
